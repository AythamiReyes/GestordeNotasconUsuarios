# GestordeNotasconUsuarios
Autor: Aythami Reyes.

Aplicacion de escritorio en Java para gestionar notas por usuarios.


Funcionalidades.

Crear, Ver, Editar y eliminar Notas.

Buscar nota por titulo.

Borrar todas las notas.

Registro de acciones por logs.

Login y registro de usuarios.

Contraseñas protegidas con SHA-256.

Los datos se guardan en ficheros.

Java con interfaz gráfica Swing. Persistencia en ficheros de texto.
